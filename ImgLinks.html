Uso de etiquetas multimedia y enlaces en HTML

Etiqueta <img>

Se usa para insertar imágenes. Es una etiqueta vacía, no tiene cierre.

Etiqueta <video>

Permite insertar videos. Se recomienda usar varios formatos para compatibilidad.

Video de YouTube

Para insertar un video de YouTube, usa la etiqueta <iframe>.

Feed de Instagram

Para insertar un feed de Instagram, usa la etiqueta <blockquote>.

Etiqueta <a>

Sirve para crear enlaces a otras páginas, documentos o ubicaciones.
